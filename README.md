# Zoho WhatsApp AI

AI-powered automation platform integrating Zoho CRM, WhatsApp workflows and OpenAI.

## Features

- Zoho CRM Lead Retrieval
- AI Lead Qualification
- WhatsApp Message Generation
- Conversation Summaries
- CRM Note Synchronization
- REST APIs with Swagger

## Tech Stack

- ASP.NET Core 8
- OpenAI
- Zoho CRM API
- C#
- Swagger
